# 园圈平台 - 部署文档 V1.0

## 环境要求

- **Node.js** >= 18
- **PostgreSQL** >= 15
- **Redis** >= 7
- **Docker** & Docker Compose (推荐)

## 快速部署（Docker）

```bash
# 1. 克隆项目
cd circle

# 2. 配置环境变量
cp .env.example .env
# 编辑 .env 修改 JWT_SECRET 和 OPENAI_API_KEY

# 3. 一键启动
docker-compose up -d

# 4. 初始化数据库（自动执行 schema.sql）
# 5. 访问 http://localhost:3000
```

## 手动部署

### 后端

```bash
cd server
cp ../.env.example .env
npm install
npm run db:migrate
npm run dev    # 开发模式 http://localhost:3001
```

### 前端

```bash
cd web
npm install
npm run dev    # 开发模式 http://localhost:3000
```

### AI Agent

```bash
cd agents/risk-match
python agent.py
```

## 关键配置

### 品牌配置
编辑 `config/brand.ts` 即可一键更换品牌名、Logo、主题色。

### Nginx 配置
编辑 `web/nginx.conf` 可配置域名、SSL、反向代理。

## 监控

- 健康检查: `GET /api/health`
- 数据库连接池: 自动重连
- 缓存命中率: Redis info

## 安全

1. 修改默认 JWT_SECRET
2. 配置 HTTPS (推荐 Let's Encrypt)
3. 开启数据库备份
4. 定期更新依赖
